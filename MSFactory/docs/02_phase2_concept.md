# Phase 2 — 이동식 컨베이어벨트 시스템 (개념)

> Phase 1 완료 후 구체화 예정. 현재는 개념 수준으로만 정리.

## 개념

바퀴 로봇들이 상황에 따라 줄을 서서 **체인 형태로 물건을 넘겨가는** 동적 컨베이어 시스템.

정적 컨베이어 벨트 없이, 로봇들이 스스로 판단하여 컨베이어 라인을 형성하고 해산.

```
일반 운반 모드:
[로봇] ──────────────────────────────> [목적지]
        물건 들고 직접 이동

컨베이어 모드 (긴 거리 or 많은 물량):
[로봇A] → [로봇B] → [로봇C] → [로봇D] → [목적지]
  물건 전달   물건 전달   물건 전달
  후 제자리   후 제자리   후 제자리
```

## 핵심 과제

- 언제 컨베이어 모드로 전환할지 판단 (거리? 물량? 로봇 가용성?)
- 로봇 간 물건 전달 타이밍 동기화
- 컨베이어 해산 후 각 로봇의 다음 태스크 배정
- 다중 에이전트 협업 RL (MAPPO 등) 필요

## 실물 구현 고려사항

- 바퀴 로봇 위에 롤러 또는 미니 컨베이어 기구 추가
- 5cm 크기 제약 내에서 물건 전달 메커니즘 설계 필요
- 로봇 간 정렬 정밀도 요구 높음
