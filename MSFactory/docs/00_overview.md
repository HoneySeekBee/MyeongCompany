# MSFactory — 프로젝트 전체 개요

## 목표

NVIDIA ISAAC Sim 기반 스마트 창고/공장 시뮬레이션을 구축하고,
강화학습(RL)으로 자율 주행 로봇을 학습시킨 뒤,
최종적으로 실물 소형 기계(개당 약 5cm)로 실제 미니 스마트공장을 구현한다.

---

## 로봇 구성

### 바퀴 로봇 (AMR) — 10대
- 물건을 한 번에 1개씩 픽업하여 목적 구역으로 운반
- 자율 주행: 경로 탐색 + 충돌 회피 (RL 학습)
- 태스크 디스패처로부터 임무 배정받아 실행

### 스테이션 로봇 (고정형) — 각 구역마다 1대
- 바퀴 로봇에 물건을 싣거나(Loading) 내리는(Unloading) 역할 둘 다 담당
- Phase 1에서는 로봇팔 또는 소형 메커니즘 (실물 구현 시 크기 고려)

---

## 개발 단계

### Phase 1 — 창고 자율 운반 시스템
- 다중 구역(A, B, C, D...) 창고 레이아웃
- 바퀴 로봇 10대, FIFO 태스크 큐 기반 자율 운반
- 상황에 따라 로봇이 동적으로 태스크 재배정
- **핵심**: 개별 로봇 RL 자율주행 + 규칙 기반 디스패처

### Phase 2 — 이동식 컨베이어벨트 시스템
- 바퀴 로봇이 상황에 따라 줄 지어 체인 형태로 물건을 넘겨가는 방식
- 정적 컨베이어 없이 로봇들이 동적으로 컨베이어 라인을 형성
- 다중 에이전트 협업 RL (MAPPO 등)

## 기술 스택

| 항목 | 기술 |
|------|------|
| 시뮬레이터 | NVIDIA ISAAC Sim |
| RL 프레임워크 | OmniIsaac Gym Envs |
| RL 알고리즘 | PPO (Phase 1), MAPPO (Phase 2) |
| 언어 | Python |
| 학습 인프라 | 로컬 RTX 4060 (개발/테스트) + RunPod (대규모 학습) |
| 실물 제어 | 미정 (ESP32 / Raspberry Pi Pico 등 검토 중) |
| 통신 | 미정 (MQTT / ROS2 검토 중) |

## 개발 환경

- OS: Windows 11 (ISAAC Sim 네이티브 설치)
- GPU: RTX 4060 Laptop 8GB (개발/씬 구축)
- CPU: Intel i7-13620H
- RAM: 64GB
- 대규모 학습: RunPod (A100 / RTX 4090 등)
