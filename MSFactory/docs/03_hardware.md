# MSFactory — 하드웨어 목록

## 개요

원룸 데스크 규모의 미니 스마트 팩토리 구축 기준.
공장 바닥 면적 약 60cm × 80cm 상정.

### 센서 배치 (로봇 1대 기준)
- VL53L0X × 3: 전방 / 좌측 / 우측 (장애물 감지)

### 섀시
- 3D 프린팅으로 제작 (약 5~6cm × 5~6cm)
- 파일 직접 설계 or Thingiverse에서 N20 모터 호환 소형 섀시 참고

## 스테이션 로봇 — 구역 수만큼

## 위치 추적 시스템

천장 카메라 + ArUco 마커 방식. 중앙 서버(랩탑)에서 OpenCV로 각 로봇 위치 실시간 파악.

| 부품 | 검색어 | 구매처 | 단가 | 수량 | 소계 |
|------|--------|--------|------|------|------|
| 웹캠 | 1080P USB 웹캠 광각 | 쿠팡 | 25,000 | 1 | 25,000 |
| ArUco 마커 | 프린터 출력 | - | 0 | 10 | 0 |

**소계: 25,000원**

### 동작 방식
1. 각 로봇 상단에 고유 ArUco 마커 부착
2. 천장 웹캠이 전체 공장 바닥 내려다봄
3. 랩탑 OpenCV가 실시간으로 각 마커 ID + 위치(x, y) + 방향 계산
4. MQTT로 각 로봇에 이동 명령 전달
5. 위치 정확도: ±1cm (충분)

## 중앙 제어 서버

별도 구매 불필요. 기존 개발 랩탑 활용.

| 역할 | 소프트웨어 |
|------|-----------|
| 위치 추적 | OpenCV + ArUco |
| 통신 브로커 | Mosquitto (MQTT) |
| 태스크 디스패처 | Python |
| ISAAC Sim 연동 | OmniIsaac Gym |

---

## 통신 구조

```
랩탑 (중앙 서버)
  ├── 웹캠 → OpenCV → 각 로봇 위치 파악
  ├── MQTT 브로커 (Mosquitto)
  │     ├── → 바퀴 로봇 10대 (WiFi, XIAO ESP32)
  │     └── → 스테이션 로봇 5대 (WiFi, XIAO ESP32)
  └── 태스크 디스패처 (Python)
```
